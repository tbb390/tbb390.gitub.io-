题型：字符串 难度：简单 2026-05-13

📖 题目大意

判断论文标题是否以 "isallyouneed" 结尾，如果是，输出 "Yes" 并打印前缀内容；否则输出 "No"。

💡 解题思路

  • 核心算法：字符串后缀匹配
  • 读取输入字符串 s
  • 判断后缀是否为 "isallyouneed"
  • 输出结果，根据是否匹配输出前缀内容
  • 时间复杂度 O(n)，空间复杂度 O(1)

⚙️ AC 代码

#include <bits/stdc++.h>
using namespace std;

int main(){
    string s;
    cin >> s;
    string tail = "isallyouneed";
    string ans;

    if(s.size() < tail.size() || s.substr(s.size() - tail.size()) != tail){
        cout << "No";
    }
    else{
        cout << "Yes" << endl;
        ans = s.substr(0, s.size() - tail.size());
        cout << ans;
    }
    return 0;
}

📎 备注 & 易错点

  • 注意字符串长度至少 13，确保不会越界
  • 后缀比较时要确保全小写
  • substr 起始下标要计算正确：s.size() - tail.size()
