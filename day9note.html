📝 刷题记录 · 第九天 UNO

题型：模拟 + 双向循环链表 难度：中等 2025-05-15

📖 题目大意

经典UNO纸牌模拟游戏，共有n名玩家围成环形坐好，每名玩家开局拥有一定数量手牌。给定m次出牌操作序列，一共四种卡牌效果：普通牌C、禁手牌S、反转方向牌R、罚摸两张牌D。玩家手牌数量归零直接出局不再参与游戏，存在禁手机制被禁玩家本轮无法出牌。按顺序输出所有玩家最终剩余手牌数量。

💡解题思路

  • 游戏规则
    玩家环形站位、出牌顺序可顺可逆、有人中途出局、存在禁手跳过出牌、每张操作牌必须依次执行不能跳过。
  • 数据结构选型
    普通数组无法快速删除出局玩家，顺序会错乱；环形结构+频繁删点最优选择双向循环链表，前驱后继快速跳转，删除操作O(1)完成。
  • 定义所需变量
    1.链表数组存储每个玩家前后位置；2.数组存每人手牌数；3.布尔数组标记是否被禁手；4.cur记录当前出牌人；5.dir记录出牌行进方向。
  • 全局执行流程
    1.初始化链表构建环形结构，初始化手牌与禁手状态；
    2.依次遍历每一张出牌；
    3.优先判断当前玩家是否被禁，被禁则解除禁手并直接跳到下一位；
    4>当前有效玩家打出一张牌，手牌数量减一；
    5.手牌清空直接从链表中移除该玩家；
    6.根据卡牌种类执行对应特效：S禁下家、R反转方向、D罚摸+禁手、C无效果；
    7.按照当前行进方向，切换到下一名出牌玩家；
  • 边界与格式处理
    遍历结束后按原始玩家序号输出手牌.
  • 复杂度分析：时间复杂度 O(n+m)，空间复杂度 O(n)，可承受2e5级别大数据范围。

⚙️ AC 代码

#include <iostream>
#include <string>
using namespace std;

const int MAXN = 200010;//题目最大数据范围2e5+10

struct Node {//双向链表结构体：pre前驱，nxt后继
    int pre, nxt;
} lst[MAXN];//开数组链表，速度比指针快

int a[MAXN];//存储每个玩家的手牌数量
bool ban[MAXN];//标记玩家是否被禁牌

inline void del(int x) {//双向链表把x从表中移除操作
    int p=lst[x].pre;
    int n=lst[x].nxt;
    lst[p].nxt=n;
    lst[n].pre=p;
}

int main() {
    ios::sync_with_stdio(false);
    cin.tie(nullptr);

    int n, m;//玩家数，出牌次数
    cin>>n>>m;

    for (int i = 1; i <= n; ++i) {//初始化每个玩家
        cin>>a[i];//读入初始手牌
        ban[i]=false;//初始都没有被禁
        lst[i].pre = i-1;//上家
        lst[i].nxt = i+1;//下家
    }

    lst[1].pre = n;//构造环形
    lst[n].nxt = 1;

    string op;//存出牌的顺序内容
    cin>>op;

    int cur = 1;//1开始出牌
    int dir = -1;//方向，1=顺时针，-1=逆时针

    for (char c : op) {//遍历第三行
        if (ban[cur]) {//禁止牌
            ban[cur] = false;
            if (dir == 1) cur = lst[cur].nxt;//顺时针下一个
            else cur = lst[cur].pre;
        }

        a[cur]--;//打出一张，手牌-1
        if (a[cur] == 0) {//win
            del(cur);//从链表删除
        }
        //各种牌的效果触发
        if (c == 'S') {//禁止牌
            int next;
            if (dir == 1) next = lst[cur].nxt;
            else next = lst[cur].pre;
            ban[next] = true;
        } 
        else if (c == 'R') {//方向牌
            dir *= -1;
        } 
        else if (c == 'D') {//+2
            int next;
            if (dir == 1) next = lst[cur].nxt;
            else next = lst[cur].pre;
            a[next] += 2;
            ban[next] = true;
        }
//普通牌没效果

        if (dir == 1) {//下一个人
            cur = lst[cur].nxt;//顺时针，下
        } else {
            cur = lst[cur].pre;//逆时针
        }
    }

    for (int i = 1; i <= n; ++i) {
        cout << a[i];
        if (i != n) cout << "\n";
    }

    return 0;
}

📎 备注 & 易错点

  • 禁手玩家仅跳过出牌动作，本轮出牌指令依旧正常消耗，不能跳过卡牌效果
  • 方向反转后，查找下家、判断顺位必须同步跟随dir变量变化
  • 玩家出局删除链表节点后，后续遍历自动跳过，无需额外判断
  • 输入输出加速语句必须加上，大数据量容易超时
  • 输出只在非最后一行换行，解决末尾空行WA问题
